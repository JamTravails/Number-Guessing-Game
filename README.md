# Number-Guessing-Game
This is a beginner number guessing game it helps beginners learn basic Java topics like generating random numbers, accepting user inputs and others.
#RULES FOR PLAYING 
you have 5 chances to guess the correct answer. each correct guess starting with the first attempt has its own score upto the fifth. losing or exceeding the number of attempts equal to zero score.
if you guess the answer in first attempt your earn 50 points and it will be decreasing by ten as the number of attempts increase i++
